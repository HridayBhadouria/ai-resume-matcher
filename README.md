AI Resume Analyzer & Job Matcher (CLI)
A CLI-based AI system which helps in matching a resume with multiple job descriptions in it by using NLP and Machine Learning.

Features
PDF/TXT resume support
NLP text preprocessing
Skill extraction
TF-IDF vectorization
Cosine similarity matching
Job ranking
Missing skill suggestions

Tech Stack
Python
scikit-learn
NLTK
PyPDF2

How to Run
pip install -r requirements.txt
python main.py resume.pdf jobs/
